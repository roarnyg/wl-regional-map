Will produce a single shortcode displaying a regional, interactive map of norway.



== Description ==

Use this shortcode to print an SVG map  of norway, with clickable sub-regions that can perform the javascript action you wish:

`[wl-regional-map region=<regionname> {action=<actionname>}]`

Default action is to perform a search for the regionname.
Two regions are supported 'nordnorge', 'sf'.

== Installation ==

Upload this plugin to your blog, place the shortcode somewhere.
